CSS - Kaskadowe arkusze stylów (ang. Cascading Style Sheets) to język służący do opisu formy prezentacji (wyświetlania) stron WWW. CSS został opracowany przez organizację W3C w 1996 r. jako potomek języka DSSSL przeznaczony do używania w połączeniu z SGML-em. Pierwszy szkic CSS zaproponował w 1994 r. Håkon Wium Lie. Arkusz stylów CSS to lista dyrektyw (tzw. reguł) ustalających, w jaki sposób ma zostać wyświetlana przez przeglądarkę internetową zawartość wybranego elementu (lub elementów) (X)HTML lub XML. Można w ten sposób opisać wszystkie pojęcia odpowiedzialne za prezentację elementów dokumentów internetowych, takie jak rodzina czcionek, kolor tekstu, marginesy, odstęp międzywierszowy lub nawet pozycja danego elementu względem innych elementów bądź okna przeglądarki. Wykorzystanie arkuszy stylów daje znacznie większe możliwości pozycjonowania elementów na stronie, niż oferuje sam (X)HTML.



  • Selektory - określają one styl znaczników HTML
    np. div { }
  • Właściwości - przedstawiają konkretne parametry stylu
    np. color
  • Wartości - opisują wartości podane do właściwości
    np. span {width: 60px;}
  • Klasy - pozwalają dodawać te same ustawienia do wielu znaczników na raz
    np. .klasa_1 {background-color: pink;}
  • Identyfikatory - działają podobnie do klas, lecz mogą być przypisane tylko do jednego znacznika
    np. #identyfikator_1 {font-size: 25pt;}
