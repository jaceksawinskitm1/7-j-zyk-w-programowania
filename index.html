C# (C Sharp; wym. [siːˈʃɑːp]; dosłownie „cis”, „c z krzyżykiem”) – wieloparadygmatowy język programowania zaprojektowany w latach 1998–2001 przez zespół pod kierunkiem Andersa Hejlsberga dla firmy Microsoft. Program napisany w tym języku kompilowany jest do języka Common Intermediate Language (CIL), specjalnego kodu pośredniego wykonywanego w środowisku uruchomieniowym takim jak .NET Framework, .NET Core, Mono lub DotGNU. Wykonanie skompilowanego programu przez system operacyjny wymaga zainstalowanego środowiska uruchomieniowego lub zawarcia go w dystrybuowanej aplikacji (tzw. aplikacja typu self-contained).



  • Dane - informacje zapisane w pamięci komputera
    np. 60 lub "abcdefg"
  • Obiekty - przechowują dane
    np. int
  • Zmienne - pozwalają przechowywać, nazywać i modyfikować obiekty
    np. int a = 50;
  • Metody - wykonują serie poleceń na zmiennych
    np. void Main() { int a += 1; }
  • Pętle i warunki - pozwalają wykonywać komendy powtarzalnie lub wykonywać je jeżeli dany warunek jest spełniony
    np. while (a < 10) {int b -= 5;}
    lub if (a == 8) { string b = "a jest równe osiem"; }
  • Klasy - mogą być instancjowane i przechowywać zmienne i metody
    np. class Klasa { void metoda() { System.Console.WriteLine("metoda została wykonana"); } int zmienna = 5; }
